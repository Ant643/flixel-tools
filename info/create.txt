
usage: create

Creates a copy of Demo project, the user will be promted to choose from a list 
if no further arguments are given.

Optional : create <value> <destination>

    <value>         : Either a string representing the Name of the Demo,
                      or a number representing the index of the Demo to create.

    <destination>   : The destination for the Demo to be created, if none given
                      the current directory will be used.
